Github博客搭建

Author: 幻心 Date: February 13, 2020  20:00:01 Category: 博客搭建

Windows环境下搭建Github博客

搭建所需软件

官方下载，默认安装：Node.js

官网下载，默认安装，可勾选在cmd中运行：Git

文本解释器

Sublime Text(推荐)或Typora

安装Hexo

Win+R打开运行窗口输入cmd，执行命令：

npm install hexo-cli -g

新建文件夹存储项目

例如我的文件夹名为Blog，在文件夹内打开cmd命令窗，执行：

hexo init . 初始化目录

hexo generate | hexo g 下载静态文件并完成配置

hexo server | hexo s 开启本地预览，本地即可访问localhost:4000预览页面

同步本地文件至Github

注册Github，创建本地仓库

仓库名字必须与注册时名字相同，后缀统一为github.io

下载github desktop客户端并登录

选择clone a respository

也可用Git 代码直接部署到Github上

在站点文件夹下编辑_config.yml进行修改

hexo generate | hexo g 编译静态文件

hexo deploy | hexo d 部署到Github

主题更改

在Blog/themes文件夹中打开cmd窗口执行：

git clone 下载主题到Blog/themes中

在Blog文件夹中打开_config.yml，修改theme值为你的主题文件名

hexo clean 清空旧网站文件`

hexo generate 生产新网站文件

hexo server 开启本地预览，浏览器打开localhost:4000

更换主题后，必须清空旧网站再生成新网站文件

博客编写

在Blog文件夹中打开cmd命令行窗口，执行：

hexo new "FileName" 新建文章

hexo new page "tags" | "about" 生成主题新标签页面

hexo generate 生成新网站

GitHub如何配置SSH Key

一、设置git的user name和email

如果你是第一次使用，或者还没有配置过的话需要操作一下命令，自行替换相应字段。

二、检查是否存在SSH Key

如果没有SSH Key，则需要先生成一下
